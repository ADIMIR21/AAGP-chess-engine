# ♟️ AAGP Chess Engine

AAGP — шахматный движок на Python с UCI-интерфейсом. Поддерживает подключение к любой UCI-совместимой GUI и играет на достойном уровне.

-----

## Возможности

- **UCI-протокол** — совместим с Arena, Cutechess, Lichess-bot и любым другим UCI-клиентом
- **Iterative Deepening** — углубляется итеративно, всегда имеет лучший ход под рукой
- **Alpha-Beta pruning** — эффективное отсечение ветвей поиска
- **Quiescence Search** — продолжает поиск в нестабильных позициях, избегая горизонтального эффекта; корректно обрабатывает шахи
- **Таблица транспозиций** — Zobrist-хеширование, записи с флагами EXACT / LOWERBOUND / UPPERBOUND, корректное хранение матовых оценок, лимит 1М записей
- **Сортировка ходов (MVV-LVA)** — TT-ход идёт первым, затем взятия по принципу «дешёвым бьёт дорогое»
- **Piece-Square Tables** — позиционная оценка для каждого типа фигур; король имеет отдельные таблицы для миттельшпиля и эндшпиля
- **Дебютная книга** — поддержка Polyglot (`book.bin`)
- **Тайм-менеджмент** — тратит ~4% оставшегося времени на ход

## Установка

```bash
pip install chess
```

-----

## Запуск

```bash
python main.py
```

Движок запускается и принимает команды по stdin в формате UCI.

### Подключение к GUI

1. Скачай [Arena](http://www.playwitharena.de/) или [Cutechess](https://cutechess.com/)
1. Engines → Add → укажи путь к `python main.py`
1. Играй

-----

## Дебютная книга (опционально)

Положи файл `book.bin` в формате Polyglot рядом с `main.py`:

```
AAGP/
├── main.py
└── book.bin
```

Если файла нет — движок думает самостоятельно с первого хода.

-----

## Архитектура поиска

```
iterative_deepening_search()
    └── minimax()  —  alpha-beta, TT, сортировка ходов
            └── quiescence_search()  —  взятия + уклонения от шахов
```

## Оценка позиции

|Фигура|Ценность|
|------|--------|
|Пешка |100     |
|Конь  |320     |
|Слон  |330     |
|Ладья |500     |
|Ферзь |900     |
|Король|20000   |

К материалу прибавляется позиционный бонус по PST. Эндшпиль определяется автоматически (отсутствие ферзей или мало фигур) — король переходит на эндшпильную таблицу и активизируется.

-----

## UCI-команды

|Команда             |Описание                                  |
|--------------------|------------------------------------------|
|`uci`               |Инициализация                             |
|`isready`           |Проверка готовности                       |
|`ucinewgame`        |Новая партия, очистка таблицы транспозиций|
|`position`          |Установка позиции (startpos / fen + moves)|
|`go wtime X btime X`|Поиск лучшего хода                        |
|`quit`              |Выход                                     |
